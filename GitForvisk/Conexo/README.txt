Este programa deve ser compilado usando os comandos do terminal do Linux:
gcc -Wall CGrafos.c FGrafos.c CGrafos.h main.c -o grafoAdjv3

Este programa deve ler o arquivo do qual o nome seja inserido quando ele peça
(formato txt, vide arquivos ja existentes para saber a estrutura) então o programa irá
imprimir a matriz adjacente do grafo. então verificará de é conexo ou não. Caso não seja,
ele ira mostrar a matriz de adjacencia dos subgrafos adquiridos do grafo principal.

Este programa NÃO funciona com grafos direcionados, não tente.
